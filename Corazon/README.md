Hallazgos Principales e Interpretación 
A través del análisis con SHAP, se determinó que los factores con mayor peso predictivo para el riesgo cardiovascular en este conjunto de datos son:

Presión Sistólica (ap_hi) y Diastólica (ap_lo): Se confirman como los predictores críticos absolutos. Rangos elevados desplazan exponencialmente la probabilidad hacia un diagnóstico positivo.

Edad: El riesgo se incrementa de forma lineal y marcada conforme aumenta la edad madura de los pacientes.

Índice de Masa Corporal (imc): La variable sintética creada demostró tener una correlación directa superior al peso o altura evaluados de manera aislada.
